REDUCTION METHOD

4x4 Rubik's Revenge

STEP 1: Solve Centers

Create a 2×2 block on each face. Remember: White is opposite Yellow.

  • Start with White, then Yellow.
  • Use inner layer moves to group center pieces.
  • Centers define your face colors!

STEP 2: Edge Pairing

Pair the split edges so they act like a single 3x3 edge.

U R U' R' U' F' U F

📌 Use this to move edges into position for pairing without breaking centers.

STEP 3: Solve as 3x3

Once edges are paired and centers are done, treat the 4x4 like a standard 3x3 cube.

  • Treat paired edges as ONE piece.
  • Ignore the inner layers during this phase.

STEP 4: Fix Parity

Parity occurs when a layer configuration is impossible on a 3x3. You need these special algorithms:

OLL Parity (One edge flipped)

r U2 x r U2 r U2 r' U2 l U2 r' U2 r U2 r' U2 r'

PLL Parity (Two edges need swapping)

2R2 U2 2R2 Uw2 2R2 Uw2

4x4 NOTATION GUIDE

R, L, U, D
Outer Layers
r, l, u, d
Inner Layers Only
Rw, Lw, Uw
Wide (2 Layers)
' / 2
Anti-clock / Double
